# Company profile schema (written to config/company.md)

```
## Firm profile
- name:
- website:
- one-line summary:
- what they do:
- deal/transaction types: [e.g. M&A sell-side, capital raise, SBA lending, recap]
- sectors / industries:
- typical deal sizes:
- typical stages:
- geography:
- positioning / edge:

## Closed transactions (cited or user-confirmed)
- [date] [type] — [parties as publicly reported] — [size if stated] — source: [URL | "user-confirmed"]

## Initial targeting read (seed for outreach-strategy)
- likely target industries / sizes / stages / company types / locations:
```

Rules: every transaction line is `source: <URL>` (public) or `source: user-confirmed`. Never list a deal without one. Round sizes to bands if exact figures aren't public.
