## Notes on rate/price language

**Hard rule:** the engine will only cite rates and prices that are written here, in plain language. It will not invent a rate. If a rate is not listed here, drafts will say "rate subject to lender pricing" or equivalent. Keep this file current — stale rates in config will appear in drafts.

**Honest rate language means:**
- State ranges, not promises ("approximately X–Y%," not "as low as X%").
- Include the honest caveat ("subject to credit profile and lender approval").
- Never imply a prospect is approved before underwriting.
