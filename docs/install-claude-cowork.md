# Install: Claude Cowork

**Dealflow** is installable from the Anthropic community marketplace directly inside Claude Cowork.

## Steps

1. Go to [claude.ai/plugins](https://claude.ai/plugins).
2. Search for **Dealflow** (publisher: Data Driven Partners).
3. Click **Install** on the `dealflow` plugin.
4. Open a Claude Cowork session. The plugin skills are now available.

## First run

Run the `setup` skill before prospecting:

> "Set up my deal-maker prospecting toolkit."

The `setup` skill will walk you through filling in each `config/` template:

- `config/company.md` — your company name, sender, address (required for CAN-SPAM)
- `config/products.md` — your offerings, fit criteria, and honest rate/price language
- `config/voice.md` — your voice rules and tone principles
- `config/proof-sources.md` — your closed deals and public data sources
- `config/dddb.md` — optional: connected data provider and compliance attestation

Setup will not declare complete until the compliance attestation in `config/dddb.md` is filled and dated (if you plan to use a connected data source).

## After setup

```
Prospect for [industry] businesses in [geography].
```

Or to use your own list:

```
Work my dormant list and draft re-engagement outreach.
```

Review the draft queue, edit any card, and approve to send. Nothing sends until you approve it.
